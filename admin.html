1단계 — data.json 다운로드

수정한 내용을 data.json 파일로 저장하세요.
이 파일을 GitHub 리포지토리에 올리면 라이브러리에 자동 반영됩니다.

2단계 — GitHub에 올리기

GitHub 리포지토리에서 data.json 파일을 클릭
연필 아이콘(편집) 클릭 → 기존 내용 전체 선택 후 삭제
다운로드한 파일 내용을 붙여넣기
Commit changes 버튼 클릭 → 완료!

처음 GitHub Pages 설정하는 방법

GitHub에서 새 리포지토리 생성 (Public으로 설정)
index.html, admin.html, data.json 3개 파일 업로드
리포지토리 → Settings → Pages → Branch: main 선택 → Save
몇 분 후 https://유저명.github.io/리포명 으로 접속 가능
